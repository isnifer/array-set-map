        const myArray = [1,2,3,4];

        const result = []

        for (let i = 0; i < myArray.length; i++) {
          // Проверяем совпадение элементов массивов

            // Если индекс текущего элемента не равен -1,

.push()/.pop() vs .unshift()/.shift()

  • .push()/.pop() — быстрые
  • .unshift()/.shift() — медленные

При использовании методов .unshift()/.shift() полностью пересчитываются все индексы в массиве, т.к. производится изменение в начале массива

Метод .reduce()

  • Принимает в качестве в аргумента функцию-итератор
  • Обходит массив с помощью итератора, возвращает нужный тип данных
  • Итератор на входе получает 4 аргумента:
    • Предыдущий результат выполнения функции или значение по умолчанию
    • Текущий элемент массива
    • Индекс текущего элемента массива
    • Массив, по которому мы пробегаемся
